# Kit - 现代化多功能工具箱

> 🧰 一个集成 70+ 种高频工具的现代化工具箱，完全本地运行，保护数据隐私

## 🌟 项目简介

Kit 是一个现代化的多功能工具集合，专为开发者、设计师和日常办公用户打造。它集成了 **70+ 种常用工具**，涵盖文本处理、颜色设计、图片音视频处理、加密哈希、数据转换等 8 大分类，让您在一个应用中完成各种任务。

### ✨ 核心特色

- **🔒 隐私安全** - 所有工具完全本地运行，数据不上传云端
- **⚡ 极速体验** - 基于现代 Web 技术，响应迅速，支持离线使用
- **📱 跨平台** - 支持 Web 端和桌面端（Windows、macOS、Linux）
- **🆓 开源免费** - MIT 开源协议，完全免费使用
- **🎨 现代设计** - 精美的 UI 设计，支持深色模式
- **🌍 国际化** - 多语言支持，本地化体验

## 📦 安装和使用

### Web 端体验

直接访问在线版本，无需安装：

```
https://kit.manon.icu/
```

### 本地开发

```bash
# 克隆项目
git clone https://github.com/aafnnp/kit.git
cd kit

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 桌面端安装

#### 开发环境

```bash
# 安装 Tauri 环境
# 参考：https://tauri.app/v2/guides/getting-started/prerequisites/

# 启动桌面端开发
npm run tauri dev
```

#### 构建安装包

```bash
# 构建桌面端应用
npm run tauri build
```

构建产物位于 `src-tauri/target/` 目录，支持：

- **Windows** - `.msi` 安装包
- **macOS** - `.dmg` 安装包
- **Linux** - `.deb` / `.rpm` 安装包
