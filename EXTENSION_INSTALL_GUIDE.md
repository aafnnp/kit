# Kit Chrome扩展安装指南

## 🎉 扩展构建成功！

Chrome扩展已成功构建完成，所有80+工具功能都已适配到浏览器扩展环境。

## 📦 构建产物

- **扩展目录**: `dist-extension/` (未压缩的扩展文件)
- **安装包**: `kit-extension.zip` (2.06 MB)

## 🚀 安装方法

### 方法一：开发者模式安装（推荐）

1. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入：`chrome://extensions/`
   - 或点击 Chrome菜单 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 在页面右上角打开"开发者模式"开关

3. **加载未压缩的扩展程序**
   - 点击"加载已解压的扩展程序"
   - 选择项目根目录下的 `dist-extension` 文件夹
   - 点击"选择文件夹"

4. **验证安装**
   - 扩展应该出现在扩展列表中
   - 浏览器工具栏会显示Kit图标

### 方法二：压缩包安装

1. 解压 `kit-extension.zip` 到本地文件夹
2. 按照方法一的步骤3-4安装解压后的文件夹

## 🎯 使用方法

### 基本操作

1. **主界面访问**
   - 点击浏览器工具栏的Kit图标
   - 或使用快捷键 Ctrl+K (Mac: Cmd+K)

2. **右键快捷访问**
   - 在网页上选中文本 → 右键 → Kit工具箱 → 文本工具
   - 在图片上右键 → Kit工具箱 → 图片工具

3. **工具导航**
   - 使用搜索框快速查找工具
   - 通过分类标签浏览工具
   - 收藏常用工具便于快速访问

### 数据管理

- **收藏功能**: 点击工具卡片的心形图标
- **最近使用**: 自动记录最近访问的工具
- **数据同步**: 配置和收藏可跨设备同步（需要Chrome登录）

## 🔒 隐私安全

- ✅ **本地处理**: 所有数据在本地处理，不上传服务器
- ✅ **最小权限**: 仅请求必要的扩展权限
- ✅ **开源透明**: 所有源代码开放可审计
- ✅ **无广告**: 扩展环境下自动隐藏广告

## 🛠 开发信息

### 技术栈

- React 18 + TypeScript
- Chrome Extension Manifest V3
- TailwindCSS + Radix UI
- Chrome Storage API

### 权限说明

- `storage`: 保存用户配置和收藏
- `activeTab`: 获取当前标签页信息
- `contextMenus`: 创建右键菜单
- `clipboardWrite`: 复制结果到剪贴板
- `notifications`: 显示操作通知

## 🐛 故障排除

### 常见问题

1. **扩展无法加载**
   - 确保Chrome版本 ≥ 88
   - 检查文件路径是否正确
   - 尝试重新加载扩展

2. **弹窗无法打开**
   - 检查popup.html和popup.js是否存在
   - 查看扩展详情页的错误信息
   - 尝试重启Chrome浏览器

3. **工具功能异常**
   - 打开开发者工具查看控制台错误
   - 检查是否有CSP策略阻止
   - 确认所需权限已授予

### 日志查看

1. 打开 `chrome://extensions/`
2. 找到Kit扩展，点击"详细信息"
3. 点击"检查视图"查看背景页面日志
4. 或在弹窗中按F12查看前端日志
